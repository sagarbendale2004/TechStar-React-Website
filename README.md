To get a local copy up and running, follow these steps:

1. **Clone the repository:**

   ```cmd

2. Navigate to the project directory:
   ```cmd
   cd xcrypto-app
   
4. Install dependencies:
   ```cmd
   bun install
   
5. Start server:
   ```cmd
   bun run dev
